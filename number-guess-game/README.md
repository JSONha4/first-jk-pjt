# 숫자 맞추기 게임
![](https://i.imgur.com/aTWSsMz.jpg)
술게임 Up Down에서 영감을 받은 프로젝트 입니다.

## 기능
* 컴퓨터는 게임이 시작할 때 랜덤한 숫자를 생성한다.
* User는 숫자를 입력할 수 있다.
* User가 입력한 숫자가 컴퓨터가 뽑은 숫자보다 작으면 Up! 이라고 알려준다.
* User가 입력한 숫자가 컴퓨터가 뽑은 숫자보다 크면 Down! 이라고 알려준다.
* User가 입력한 숫자가 컴퓨터가 뽑은 숫자와 일치한다면 "정답!" 이라고 뜨고 게임이 종료된다.
* User는 총 5번의 기회가 있다.
* 게임이 종료되면 Start 버튼은 비활성화 된다.
* Reset 버튼을 누르면 게임이 초기화 된다.
* User가 1 ~ 100 범위 밖에 숫자를 입력할 시에 경고 메세지가 뜬다.
* User가 이미 입력한 값을 또 입력할 시에 경고 메세지가 뜬다.
* 반응형 UI

## 사용한 기술
* HTML
* CSS
* Vanila Javascript
